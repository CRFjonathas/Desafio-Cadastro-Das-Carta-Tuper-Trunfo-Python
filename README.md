# 🃏 Super Trunfo - Países (Versão Python)

## 🎯 Descrição
Este repositório contém a recriação da minha primeira atividade de programação da faculdade, originalmente desenvolvida em C, agora reescrita em **Python**. O projeto implementa a base do clássico jogo "Super Trunfo", com o tema Países/Cidades. 

O foco principal é praticar a entrada e saída de dados, conversão de tipos, operações matemáticas e estruturas de decisão, evoluindo do cadastro simples até a comparação direta entre atributos de diferentes cartas.

## 🚀 Tecnologias
* ![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)

## 📂 Funcionalidades Implementadas

O projeto foi construído seguindo a progressão lógica de 3 níveis:

* **Nível Novato (Fundamentos):** * Cadastro interativo de cartas (Estado, Código, Nome, População, Área, PIB, Pontos Turísticos).
    * Exibição formatada dos dados lidos.
* **Nível Aventureiro (Processamento):** * Cálculos matemáticos automáticos implementados durante o cadastro:
        * **Densidade Populacional:** (População / Área).
        * **PIB per Capita:** (PIB / População).
* **Nível Mestre (Lógica Condicional):** * Cálculo do "Super Poder" (soma de todos os atributos).
    * Algoritmo de comparação entre duas cartas.
    * Regra de negócio invertida aplicada para a *Densidade Populacional* (onde o menor valor vence).
